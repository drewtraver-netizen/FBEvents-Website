Getting Started

  1. Log In — Click "Log In to Facebook." A browser opens. Log into your Facebook account. Your session is saved.
  2. Detect Groups — Click "Detect My Groups." All your groups appear. Check the ones you want.
  3. Scan Events — Click "Scan Selected Groups." Upcoming events populate the table. Sort by clicking column headers.
  4. Export — Click "Export" and save as HTML. Open the file — every URL is a clickable link.

Frequently Asked Questions

Why does the app need to open a browser?

Facebook loads its content dynamically with JavaScript. A real browser is required to render the pages and extract events. That's also why the app includes Chromium (about half the file size).

How do I see events for only some of my groups?

After clicking "Detect My Groups," each group appears in the left panel with a checkbox. Uncheck any groups you don't want to scan. Only checked groups will be scanned.

The date column shows YYYY-MM-DD, not "May 9"

Dates in the table are stored in YYYY-MM-DD format for proper chronological sorting. The original display date (like "May 9") appears in the exported HTML file.

Groups show numeric IDs instead of names

Run "Detect My Groups" again. This can happen if Facebook's page structure changes between sessions.

My login session expired

Go to Account → Log Out, then click Log In to re-authenticate. The app saves your new session for reuse.

Does this violate Facebook's terms?

Facebook Group Events Summarizer automates what you can already do manually — visit each group's events page and note the dates. It does not modify or bypass anything. Use responsibly.
